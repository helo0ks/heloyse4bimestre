# Relatório do Projeto – Loja de Itens do Snoopy

Meu projeto é uma loja de itens do Snoopy, atualmente apenas com pelúcias.

Com ele, aprendi principalmente:

- A lógica da programação.  
- A separação entre **backend** e **frontend** e sua importância.  
- Como seguir uma formatação específica, baseando-me no projeto base apresentado.  
- Diferentes formas de resolver o mesmo problema, como salvar imagens via *localStorage*, salvar na máquina ou enviar para o banco de dados.

Apesar disso, não obtive muito conhecimento da linguagem devido ao uso de IA.  
Consigo compreender tudo que acontece no sistema, porém nunca conseguiria fazer meu site manualmente.

## Dificuldades

Minha principal dificuldade foi no terceiro bimestre, para entender e colocar em prática os CRUDs adaptados para o meu banco de dados e como funcionava a ideia de definir uma pessoa como cliente ou admin.

Consegui superar isso assistindo vídeos no YouTube e com ajuda em um PA. A partir do momento em que consegui adaptar o CRUD de **cargo** para o meu projeto e banco de dados, foi muito mais simples alterar os outros.

Parte dessa dificuldade também foi culpa da IA, que não queria seguir a ideia do projeto base (separação de *routes*, *controller* e frontend). Quando resolvi manualmente o primeiro CRUD, usar a IA para modificar os outros se tornou mais simples.

## Resultados

No geral, consegui entregar tudo o que esperava do meu projeto.  
O funcionamento ficou bem dinâmico, e esteticamente também me agradou.

## Uso de IA

Como expliquei anteriormente, em alguns momentos a IA mais atrapalhou do que ajudou. Porém, quando a base já estava consolidada, fazer alterações se tornou muito mais simples.

No quarto bimestre, usei o **Copilot no modo agente** utilizando a **IA Claude Opus 4.5**, e entre todas que testei, foi a mais eficiente e rápida na resolução dos problemas.

Além disso, usei o **ChatGPT para criar prompts usando frameworks**, e isso com certeza foi essencial para a execução.

## Preferência Profissional

Tenho preferência pela área de **backend**. Desde o início da matéria de banco de dados, tive mais facilidade e afinidade. Já tive vontade de seguir me especializando na área, mas atualmente não me vejo atuando em TI.
